## 1. このアプリが実際に扱っているデータ

コード（`js/auth.js`, `js/api.js`）を確認した結果、収集・保存しているのは以下。
これがプライバシーポリシーに書くべき内容の土台になる。

- **メールアドレスとパスワード**
  - `js/auth.js` の Firebase Authentication で、ログイン認証に使用
  - パスワードはFirebase Authenticationに送られ、ハッシュ化されて保存されている
  - メールアドレスはログイン時の本人特定用に使用。Googleインフラ上で保存時に暗号化された状態で保存される。
- **コーヒーの記録データ**
  - `js/api.js` で Firestore に保存
  - 具体例：商品名 / 生産国 / 農園 / 品種 / プロセス / 風味 / ドリッパー / レシピ / 購入店 / メモ / 味の評価（酸味・苦味・コク・甘味・香り）
- これらは **Google の Firebase**（外部サービス）のサーバー上に保存されている

### 用語メモ
- **Firebase（ファイアベース）**: Googleが提供するアプリ開発基盤。
- **Firestore（ファイアストア）**: Firebaseの中のデータベース機能。
- ユーザーのデータは自分のPCではなく、Googleのサーバーに預けて保管している。
  → 「第三者（Google）にデータを預けている」事実は明示が必要。
